# 社群软件 APP 测试需求

## 一、测试目标

对 APP 的核心功能、页面流程、数据准确性、稳定性、安全性进行完整测试，确保用户可以正常注册、登录、聊天、加入社群、发布内容、管理资料和接收通知。

## 二、测试范围

### 1. 登录 / 注册模块

测试内容：

- 手机号注册
- 验证码获取
- 密码登录
- 忘记密码
- 退出登录
- 自动登录
- 用户协议 / 隐私政策跳转

重点检查：

- 手机号格式校验是否正确
- 验证码倒计时是否正常
- 密码错误是否有提示
- 账号被封禁是否限制登录
- 退出后是否清除登录状态

### 2. 首页 / 社群列表

测试内容：

- 推荐社群展示
- 我的社群展示
- 社群搜索
- 社群分类筛选
- 社群详情页
- 加入 / 退出社群

重点检查：

- 社群列表是否加载正常
- 无网络时是否有提示
- 搜索结果是否准确
- 加入社群后状态是否变更
- 退出社群后是否从“我的社群”移除

### 3. 聊天模块

测试内容：

- 单聊
- 群聊
- 发送文字、图片、语音、视频、文件、表情
- 消息撤回
- 消息删除
- 消息已读 / 未读
- 历史消息加载

重点检查：

- 消息发送是否实时
- 对方是否能收到
- 断网重连后消息是否补发
- 图片 / 视频是否正常预览
- 大文件发送是否有限制
- 消息顺序是否正确
- 撤回消息是否双方同步
- 未读数量是否准确

### 4. 社群内容发布

测试内容：

- 发布文字、图片、视频动态
- 删除动态
- 点赞
- 评论
- 回复评论
- 举报内容

重点检查：

- 发布后是否即时展示
- 图片数量限制是否生效
- 视频大小限制是否生效
- 删除后是否所有端同步消失
- 点赞数 / 评论数是否准确
- 敏感词是否拦截
- 举报后后台是否收到记录

### 5. 用户资料模块

测试内容：

- 修改头像
- 修改昵称
- 修改简介
- 修改性别
- 修改生日
- 查看他人主页
- 拉黑用户
- 关注 / 取消关注

重点检查：

- 头像上传是否成功
- 昵称长度限制是否正常
- 特殊字符是否过滤
- 拉黑后是否无法私聊
- 关注状态是否同步

### 6. 通知模块

测试内容：

- 系统通知
- 评论通知
- 点赞通知
- 私信通知
- 社群公告通知
- 推送通知

重点检查：

- APP 前台是否收到通知
- APP 后台是否收到推送
- 点击通知是否跳转正确页面
- 未读红点是否准确
- 已读后红点是否消失

### 7. 社群管理模块

测试内容：

- 创建社群
- 修改社群名称、头像、简介
- 设置入群方式
- 管理成员
- 踢出成员
- 禁言成员
- 设置管理员
- 发布公告

重点检查：

- 普通成员是否无管理权限
- 管理员权限是否正确
- 被踢用户是否无法继续发言
- 禁言用户是否无法发消息
- 公告是否所有成员可见

### 8. 搜索模块

测试内容：

- 搜索用户
- 搜索社群
- 搜索聊天记录
- 搜索动态内容

重点检查：

- 关键词搜索是否准确
- 空关键词是否限制
- 无结果是否有空状态
- 搜索历史是否记录
- 是否支持清空搜索历史

### 9. 安全测试

测试内容：

- 未登录访问接口
- 越权操作
- 敏感词过滤
- XSS 文本注入
- SQL 注入
- 文件上传类型限制
- 多端登录状态

重点检查：

- 普通用户不能调用管理员接口
- 被禁言用户不能绕过发言
- 上传文件不能伪装成图片
- 登录 token 过期后是否强制重新登录

### 10. 兼容性测试

测试设备：

- Android 8 / 9 / 10 / 11 / 12 / 13 / 14 / 15
- 不同屏幕尺寸
- 刘海屏 / 挖孔屏
- 低端机 / 中端机 / 高端机

重点检查：

- 页面是否变形
- 按钮是否遮挡
- 输入框是否被键盘挡住
- 图片是否拉伸
- 滚动是否卡顿

## 三、测试重点场景

| 场景 | 测试要求 |
| --- | --- |
| 新用户注册 | 注册、登录、完善资料、加入社群 |
| 老用户登录 | 登录后历史消息、社群、通知是否正常 |
| 群聊发消息 | 多人同时发消息，顺序不能乱 |
| 断网重连 | 断网期间提示，恢复后自动同步 |
| 多端登录 | 手机 A/B 同时登录，消息状态一致 |
| 权限测试 | 普通用户不能管理社群 |
| 文件上传 | 图片、视频、语音、文件限制正确 |
| 推送测试 | 前台、后台、锁屏状态通知正常 |

```text

APP版本：

预期结果：

```

严重等级：

```text
S1：阻塞问题，无法登录、无法使用核心功能
S2：严重问题，聊天、社群、支付等核心功能异常
S3：普通问题，页面展示、提示、交互异常
S4：轻微问题，文案、样式、边距问题
```

## 五、验收标准

- 注册 / 登录流程正常
- 聊天消息稳定收发
- 社群加入、退出、管理正常
- 图片、视频、文件上传正常
- 通知和红点数量准确
- 无明显页面错位
- 无严重闪退、卡死
- 无明显权限漏洞
- 核心功能在主流 Android 机型正常运行
